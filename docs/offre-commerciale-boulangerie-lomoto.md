# Offre commerciale - Boulangerie Lomoto

## Objet

Fourniture, installation et mise en service d'une application Windows de gestion complète pour boulangerie.

## Solution proposée

L'application permet de gérer les opérations essentielles d'une boulangerie :

- commandes des clients, mamans, ventes cash et dépositaires ;
- caisse, entrées, dépenses, dettes payées et solde journalier ;
- stock, approvisionnements et sorties ;
- production journalière ;
- commissions ;
- utilisateurs et droits d'accès ;
- rapports PDF et Excel ;
- tableau de bord avec indicateurs clés ;
- fonctionnement en mode connecté entre un serveur principal et les postes clients.

## Composition de l'offre

Cette offre comprend :

- licence complète de la version Windows ;
- installation sur 1 poste serveur utilisé par l'administrateur ;
- installation sur 3 postes clients ;
- configuration du mode connecté ;
- formation des utilisateurs ;
- assistance au démarrage ;
- maintenance et support gratuits pendant les 6 premiers mois.

## Maintenance et support

Pendant les 6 premiers mois, le support comprend :

- assistance en cas de difficulté d'utilisation ;
- correction des anomalies liées à l'application ;
- aide à la configuration réseau ;
- accompagnement des utilisateurs ;
- vérification du bon fonctionnement du serveur et des postes clients.

Après les 6 mois gratuits, un contrat de maintenance peut être proposé séparément.

## Limites de cette offre

Cette offre concerne uniquement la version Windows.

Elle ne comprend pas :

- version Web ;
- version APK Android ;
- accès à distance depuis une autre connexion Internet sans configuration complémentaire ;
- achat des ordinateurs, imprimantes ou matériels informatiques ;
- frais d'internet ou de réseau ;
- développements spécifiques non prévus dans cette offre ;
- déplacements hors zone convenue.

Les versions Web et Android peuvent faire l'objet d'une offre complémentaire.

Le mode connecté Windows est prévu pour un réseau local interne. Un accès depuis une autre connexion Internet nécessite une solution distante complémentaire comme VPN, Tailscale, ZeroTier ou hébergement en ligne.
